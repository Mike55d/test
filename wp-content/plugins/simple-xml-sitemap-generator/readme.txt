== Description ==
XML Sitemap creates an XML for use with Google and Yahoo (and Yes! Bing too)
Just install it to your wordpress installation and let the plugin do his job.

For start: create or update any Post or Pages and your WordPress will create your sitemap.xml right away

= INFORMATION =
New: 
If you want you can now add your post category to your sitemap

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip the Compressed file.
2. Upload `automatic-sitemap` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in Dashboard.

== Frequently Asked Questions ==

= How to install this plugin? =

Just follow this Steps
1. Unzip the Compressed file.
2. Upload `automatic-sitemap` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in Dashboard.

= How Can I access my sitemap? =

After install the plugin, simply create (or Update one) and WordPress create your sitemap.xml right away. Then you will able to access your sitemap in http://yoursitelink.com/sitemap.xml
